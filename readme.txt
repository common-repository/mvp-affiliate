## Description
MVPAffiliate is a WordPress plugin that provides an easy way to attach your Contributor ID to Microsoft affiliate links.

## Installation
1. Activate the plugin through the 'Plugins' menu in WordPress.
2. Go into your settings, which should have a new section for your MVPPlugin settings
3. Insert your MVP ID as the prompt explains and save
4. Enjoy!
